Nature of Code Exercises

Interactive generative-art projects built with JavaScript and p5.js, exploring physics and generative systems, based on concepts from The Nature of Code by Daniel Shiffman. Hope you like them!

Chapter Project Description
Randomness Walker Lab An exploration of randomness and motion, showcasing multiple probabilistic movement systems that generate real-time random walk patterns, ranging from structured directional steps to smooth noise-driven motion.
Gaussian Spray Can Digital simulation spray painting using Gaussian distributions to control position, size, and color variation. You can paint directly with the cursor and use sliders to adjust dispersion, hue, saturation, and luminance.
Procedural Sky A slow-moving 2D cloudy sky simulation created with procedural noise and dynamic color mapping, designed to evoke a calm, atmospheric mood. Paired with ambient piano music and a minimalist interactive interface.
